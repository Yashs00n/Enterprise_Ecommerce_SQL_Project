# Enterprise E-Commerce SQL Project

## 📌 Project Overview

This project is a complete Enterprise-Level E-Commerce Database built using MySQL.

The objective of this project is to learn SQL from Beginner to Advanced by implementing real-world business scenarios.

This project follows industry-standard database development practices.

---

## 🎯 Objectives

- Learn SQL from Beginner to Advanced
- Design Enterprise Database
- Create Master & Transaction Tables
- Implement Constraints
- Perform CRUD Operations
- Write Complex SQL Queries
- Learn Joins
- Learn Subqueries
- Learn CTE
- Learn Window Functions
- Learn Views
- Learn Indexes
- Learn Stored Procedures
- Learn Triggers
- Create Business Reports
- Optimize SQL Queries

---

## 🛠 Technology Stack

Database : MySQL 8

Version Control : Git & GitHub

IDE : GitHub.dev

Query Execution : DB Fiddle / SQLize.online
